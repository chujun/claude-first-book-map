豆瓣作者页面的信息通过 **JavaScript 动态渲染**，传统 `requests` 无法获取。需要使用 Playwright 模拟浏览器行为。

### 3.3 作者信息提取逻辑

```python
def fetch_author_info(self, url: str) -> Optional[Dict]:
    """
    从作者页面提取信息
    返回: {
        'name': '作者名',
        'gender': '男/女',
        'birth_date': '1964年9月',
        'country': '中国',
        'birthplace': '山西,阳泉'
    }
    """
    result = {
        'name': '',
        'gender': '',
        'birth_date': '',
        'country': '',
        'birthplace': ''
    }

    # 1. 从 h1 提取作者名
    h1 = page.locator('h1')
    if h1.count() > 0:
        result['name'] = h1.inner_text().split()[0]

    # 2. 从 span 元素提取位置信息
    # 格式: "中国,山西,阳泉"
    spans = page.locator('span').all()
    for span in spans:
        text = span.inner_text().strip()
        # 排除版权信息和无效文本
        if ',' in text and not text.startswith('©'):
            parts = [p.strip() for p in text.split(',')]
            if len(parts) >= 1:
                result['country'] = parts[0]  # 第一部分是国家
            if len(parts) >= 2:
                result['birthplace'] = ','.join(parts[1:])  # 剩余是出生地
            break

    # 3. 提取性别
    for span in spans:
        text = span.inner_text().strip()
        if text == '男' or text == '女':
            result['gender'] = text
            break

    # 4. 提取出生日期
    for span in spans:
        text = span.inner_text().strip()
        if re.match(r'\d{4}年\d{1,2}月\d{0,2}日?', text):
            result['birth_date'] = text
            break

    return result if result['name'] else None
```

### 3.4 关键解析规则

| 信息类型 | 解析位置 | 解析方法 | 示例 |
|---------|---------|---------|------|
| 作者姓名 | `<h1>` 标签 | `inner_text().split()[0]` | "伊塔洛·卡尔维诺" |
| 性别 | `<span>` 文本 | 精确匹配"男"/"女" | "男" |
| 出生日期 | `<span>` 文本 | 正则匹配日期格式 | "1923年10月" |
| 国家 | `<span>` 文本 | 逗号分隔第一部分 | "中国" |
| 出生地 | `<span>` 文本 | 逗号分隔剩余部分 | "山西,阳泉" |

## 5. 数据质量保障

### 5.1 数据一致性修复

```
┌─────────────────────────────────────────────────────────────────┐
│                    数据一致性修复流程                              │
└─────────────────────────────────────────────────────────────────┘

检测同一作者的不同书籍是否存在数据不一致:

   作者: "余华"
   ├── 书籍1: 性别='男', 生日='1960年4月'
   ├── 书籍2: 性别='',  生日='1960年4月'  ← 缺失
   └── 书籍3: 性别='男', 生日=''

修复逻辑:
   UPDATE books
   SET author_gender = '男'
   WHERE author = '余华'
   AND (author_gender IS NULL OR author_gender = '')
```

### 5.2 坐标合理性校验

| 检查规则 | 处理方式 |
|---------|---------|
| 中国作家坐标应在: lat∈[15,55], lng∈[70,140] | 超范围标记为异常 |
| 同一城市的作家应有相近坐标 | 差异过大时人工核查 |
| 首都坐标作为国家默认值 | 确保每个国家都有基础坐标 |

### 5.3 数据覆盖率统计

| 数据字段 | 覆盖率 | 缺失原因 |
|---------|-------|---------|
| 坐标 (lat/lng) | 73% | 匿名作品/作者信息缺失 |
| 国家 | 73% | 同上 |
| 性别 | 89% | Douban未收录 |
| 出生日期 | 65% | 历史人物/未收录 |
| 评分 | 70% | 部分书籍无评分 |

## 7. 常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 作者URL在#info区域找不到 | 豆瓣页面结构变化 | 添加全页面fallback搜索 |
| 版权信息被误解析为位置 | 版权文本含逗号 | 过滤`startswith('©')` |
| 中国城市全部解析为北京 | 城市名含省份前缀 | 添加`,`分隔解析 |
| 台湾作家无坐标 | 国家名不匹配 | 添加"中国台湾"别名映射 |
| 坐标为(0,0) | 解析失败 | 回退到国家首都坐标 |
